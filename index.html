Is the weather the same in New York and LA?

http://istheweatherthesameinnewyorkandla.com

Is the weather the same in New York and L.A.? I built this single-serving site to help answer this age-old question. The app uses an openweathermap.org API call to determine weather conditions and temperature, then compares the data. Finally, the app returns one of the following responses: "Yes", "Pretty close", "Sort of", "Not really" or "Nope" depending on both cities' weather conditions.

Technology used: JS, jQuery API call, CSS, HTML

  • Client: Start Bootstrap
  • Date: April 2015
  • Service: Web Development
